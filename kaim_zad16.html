Opis znacznika DIV (jaki to element, składnia w ramce, zastosowanie, opis treotyczny, atrybuty tylko wymień, opisz tylko ALIGN).

Element HTML ogólnego przeznaczenia, grupujący inne elementy HTML w tzw. sekcje.

align okresla w jakim miejscu na stronie ma się znajdowac

Opis znacznika SPAN (jaki to element, składnia w ramce, zastosowanie, opis treotyczny).

HTML < span > element jest podstawowym wbudowanym kontenerem dla frazowania treści, który z natury nie reprezentuje niczego. Może być używany do grupowania tekstu w celu stylizacji (używa atrybutu class lub id), lub z powodu posiadania takich samych atrybutów, takich jak lang. Powinien być używany tylko wtedy, gdy nie ma zastosowania żaden inny element semantyczny.

Różnica między znacznikami DIV i SPAN.

< span > jest bardzo podobny do elementu < div >, lecz < div > jest elementem blokowym, a < span > - elementem liniowym.

Opisz oraz wymień elementy/tagi: 1)liniowe 2)blokowe 3)blokowo-liniowe.

Elementy liniowe

a, bdo, br, img, map, object, q, script, span, sub, sup

Elementy blokowe

p, h1, h2, h3, h4, h5, h6, ol, ul, div, table

Elementy blokowo-liniowe

input, textarea
